We use static IPs

You are probably used to simply plugging one end of an ethernet cable into your router and the other end into the ethernet jack and just like that you can browse the internet. This relies on a protocol called DHCP that automatically provides an IP address and other access data to your router. We have not (yet) implemented this service so you need to manually configure this data. Most routers allow this configuration under a menu entry called "WAN" or "Internet". If this is not the case for your router model, please consult the manual.

Troubleshooting

If your internet was working before and suddenly stopped working, troubleshoot the problem with the following steps:

  1. make sure your account balance is not negative
  2. restart the router
  3. check if the static IP configuration is correct
  4. verify the cable is okay, try a different one if available
  5. reset the router and enter the static IP configuration again
  6. try connecting a computer directly (make sure to configure the static IP on the computer)
